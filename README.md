# Промежуточное ПО

## Установка и запуск

Установка

```
npm install
```

Запуск

```
npm start
```

## Использование

Получение главной страницы

```
http://localhost:5000/
```

Получение суммы чисел

```
http://localhost:5000/sum/find?a=12&b=8
```

При попытке получить несуществующую страницу выходит страница с сообщением "404"

```
http://localhost:5000/aaa/bbb/ccc/xxx
```
